DATA SOURCES

Where to Get Satellite Images

Use these official, free sources to download Landsat 8/9 IR and RGB band data for training and testing.

PRIMARY

USGS EarthExplorer

Official USGS portal for Landsat 8/9 Collection 2 data. Download Band 10/11 (TIRS thermal) and Bands 4/3/2 (RGB).

1 Go to earthexplorer.usgs.gov
2 Select "Landsat" → "Landsat 8-9 OLI/TIRS C2 L2"
3 Draw AOI on map, set date range
4 Download Band 10 (IR) + Bands 4,3,2 (RGB)
Visit EarthExplorer →
ALTERNATIVE

Google Earth Engine

Cloud-based platform to access, process, and export Landsat collections at scale via Python API.

1 Sign up at code.earthengine.google.com
2 Use ee.ImageCollection('LANDSAT/LC09/C02/T1_L2')
3 Select bands ST_B10 (thermal) + SR_B4/B3/B2
4 Export to Google Drive as GeoTIFF
Visit GEE →
BENCHMARK

Copernicus Open Hub

ESA's free hub for Sentinel-2 multispectral data, useful as supplementary paired training data.

1 Visit scihub.copernicus.eu
2 Search for Sentinel-2 L2A tiles
3 Use B8A (NIR) as IR proxy
4 B4/B3/B2 for true color RGB
Visit Copernicus →

EVALUATION METRICS

How We Measure Quality

PSNR

Peak Signal-to-Noise Ratio

Measures reconstruction fidelity in decibels. Higher is better. Target: ≥ 35 dB.

Target: 42 dB
SSIM

Structural Similarity Index

Compares luminance, contrast, and structure between output and ground truth. Range: 0–1.

Target: 0.97
FID

Fréchet Inception Distance

Evaluates visual realism by comparing feature distributions. Lower is better.

Target: <30
⚡ T

Inference Time

Time to process one full satellite tile (256×256 px). Ensures production scalability.

Target: <2s
